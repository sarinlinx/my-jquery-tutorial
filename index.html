Effects

    • List of jQuery Effects
    • hide() - hides an element
    • show() - displays a hidden element
    • toggle() - Toggles between the hide() and show() methods
    • slideDown() - Slides (and displays) the selected hidden element down on the page
    • slideUp() - Slides (and hides) the selected displayed element up on the page
    • slideToggle() - Toggles between slideUp() and slideDown() for the selected elements
