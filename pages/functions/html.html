Changing HTML using the html() method

The html() method changes the HTML of a specified element

It also adds a CSS class to the element.

<p class="dope"> You'll learn how to replace me</p>
<input type="button" value="Replace" id="btn_main" />

$('#btn_main').on('click', function() {
  $('.dope').html('<strong class="html-change-class">l; HI! The HTML just changed. </strong>');
});
