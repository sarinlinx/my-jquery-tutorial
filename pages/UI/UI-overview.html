Keyup Content Match in form field

keyup() is used to check the 2nd input field.

If something is entered, the function named 'validate' is run

The HTML of the page is then updated.

  • Example 1
<div id="output">Output1 Content</div>
<!-- input fields -->
<input id="pswa" class="matchField" type="text" />
<br />
<input id="pswb" class="matchField" type="text" />
<div id="status"></div>
                               
 $(document).ready(function() {
  // only checking pswb because it's assumed the first field pswa is already filled in
  // on keyup is triggered, validate function is run
  $("#pswb").keyup(validate);

  function validate() {
    // when function is triggered, the 'status' div changes to 'Working'
    $("#status").text("Working");
  }
})

Add comparison check of 2 fields

Create vars in the function to obtain the value of each field.

Create if else statment to check if they match.

CSS is also removed and added using 'removeClass' and 'addClass'.

  • Example 2
.highlight {
  background-color: red;
}
                  
<div id="output">Output1 Content</div>
<!-- input fields -->
<input id="pswa" class="matchField" type="text" />
<br />
<input id="pswb" class="matchField" type="text" />
<div id="status"></div>
                               
 $(document).ready(function() {
  // only checking pswb because it's assumed the first field pswa is already filled in
  // on keyup is triggered, validate function is run
  $("#pswb").keyup(validate);

  function validate() {
    // obtain each value of the fields
    var psw1 = $("#pswa").val();
    var psw2 = $("#pswb").val();

    // run a comparison of the 2 fields
    if (psw1 === psw2) {
      $("#status").text("MATCHED");
      // remove the 'highlight' class
      $(".matchField").removeClass("highlight");
    } else {
      $("#status").text("NOT MATCH");
      // add the highlight class
      $(".matchField").addClass("highlight");
    }
  }
})
