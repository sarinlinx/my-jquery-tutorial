prepend()

Same as .append(), but it adds to the beginning

.after()

This adds the elmeent after the targeted element

.before()

Adds the element before the targeted element

remove()

The hide() method just adds a style attribute so youcannot see the Object, but it's still there.

The remove() method removes the Object from the page
