Target only the list elements in a list

Target the ID of the list, add a space, then select the li items.

<ul id="steps">
  <li>Step 1</li>
  <li>Step 2</li>
</ul>

$('#steps li').css('color', 'yellow');

Target a nested list (direct descendants)

Target the direct descendents using the > sign.

This ensures only the immediate descendants of the element with an ID of steps are targeted.

<ul id="steps">
    <li>Step 1</li>
    <li>Step 2</li>
      <ul id="nested_steps">
        <li>nested item 1</li>
        <li>nested item 1</li>
      </ul>
</ul>

$('#steps > li').css('color', 'yellow');

Target only the first item in a list using a Filter

The filter would be li:first. This targets the first list item.

You could also use li:last

<ul id="steps">
  <li>Step 1</li>
  <li>Step 2</li>
    <ul id="nested_steps">
      <li>nested item 1</li>
      <li>nested item 1</li>
    </ul>
</ul>

$('#steps > li:first').css('color', 'yellow');

Target all elements that contain a specific attribute

You can target all elements that contain a specific attribute using []

For example, to target all paragraphs that have a class assigned to them use p[class].

<p class="class1">I'm a paragraph with a class</p>
<p class="class2">Me too!</p>
<p>I don't have a class</p>

$('p[class]').css({'border': '1px solid red'});

Target all elements that contain a specific attribute and name

This targets a specific attribute with a specific name

Notice how the paragraph with a class of 'class4' does not show.

You MUST USE double quotes to surround the element and class names. For example

    Yes
  • $("p[class='class4']").hide();
    • No
  • $('p[class='class4']').hide();

It will not work if you use single quotes.

<p class="class3">I'm a paragraph with a class</p>
<p class="class4">Me too!</p>
<p>I don't have a class</p>

$("p[class='class4']").hide();

Target paragraph siblings using ~

This targets the paragraph siblings of a div

sel_previous_siblings.asp

You shouldn't use this since you can't pinpoint a specific element with an ID or Class. You can only use this with the normal element tag such as 'div', not 'div#className'.

Target immediately adjacent siblings with +

This targets the immediate next element after the div..

<p>I'm a paragraph just before the div.</p>
<div id="divsibs">
  <p>I'm a paragraph in a div</p>
</div>
<p>I'm a paragraph just after the div.</p>
<p>I'm the 2nd paragraph after the div.</p>

$("#divsibs + p").css('color', 'yellow');
