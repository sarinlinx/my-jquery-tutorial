Target only the list elements in a list

Target the ID of the list, add a space, then select the li items.

<ul id="steps">
  <li>Step 1</li>
  <li>Step 2</li>
</ul>

$('#steps li').css('color', 'yellow');

Target a nested list

Target the direct descendents using the > sign.

This ensures only the immediate descendants of the element with an ID of steps are targeted.

<ul id="steps">
    <li>Step 1</li>
    <li>Step 2</li>
      <ul id="nested_steps">
        <li>nested item 1</li>
        <li>nested item 1</li>
      </ul>
</ul>

$('#steps > li').css('color', 'yellow');

Target only the first item in a list using a Filter

The filter would be li:first. This targets the first list item.

You could also use li:last

<ul id="steps">
  <li>Step 1</li>
  <li>Step 2</li>
    <ul id="nested_steps">
      <li>nested item 1</li>
      <li>nested item 1</li>
    </ul>
</ul>

$('#steps > li:first').css('color', 'yellow');

Target only the first items in both lists

Separate call to each item with a comma

<ul id="steps">
<li>Step 1</li>
<li>Step 2</li>
  <ul id="nested_steps">
    <li>nested item 1</li>
    <li>nested item 1</li>
  </ul>
</ul>

$('#steps li:first, #nested_steps li:first').css('color', 'yellow');

Target specific list item

Use nth-child(#) to target

This selects the 3rd item

<ul id="steps">
<li>Step 1</li>
<li>Step 2</li>
<li>Step 3</li>
<ul id="nested_steps">
  <li>nested item 1</li>
  <li>nested item 1</li>
</ul>
</ul>

$('#steps li:nth-child(3)').css('color', 'yellow');

Target all elements that contain a specific attribute

You can target all elements that contain a specific attribute using []

For example, to target all paragraphs that have a class assigned to them use p[class].

<p class="class1">I'm a paragraph with a class</p>
<p class="class2">Me too!</p>
<p>I don't have a class</p>

$('p[class]').css({'border': '1px solid red'});
